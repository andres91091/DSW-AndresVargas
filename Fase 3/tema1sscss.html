Sintaxis CSS


Existen 3 tipos de declaraciones en CSS, la primera se conoce como CSS en línea y hace referencia a las declaraciones de propiedad que se hace dentro de la misma etiqueta HTML.
Por ejemplo:

<h1 style="text-align:center"> nombre del título </h1>

Con esto logramos que el titulo de referencia h1 se ubique en la posición central de la página.

La segunda declaración se conoce incrustado y hace referencia a que las propiedades que queremos que tomen las diferentes variables las declaramos en el comienzo de la pagina en el espacio denominado “HEAD”
Por ejemplo:

<head> <style> h1 { padding: 20px } </style> </head>

Con esto logramos que un titulo h1 tenga un espacio de 20 magapixeles.

Por último tenemos la declaración externa, en este tipo de declaración debemos crear un archivo con extensión .css y debemos enlazarlo con nuestra página en HTML en dicho archivo crearemos las diferentes propiedades que queremos aplicar y nuestra página las tomara.
Lo primero será enlazar nuestro archivo creado con nuestra pagina y para eso usaremos la etiqueta link
Por ejemplo:

<head> <link rel="stylesheet" href="css/editores.css”>

De esta manera logramos enlazar nuestro archivo con extensión .css llamado editores ubicado en la carpeta css.
Ahora vamos a crear en nuestro archivo .css una declaración para darle unas propiedades a una etiqueta
Ejemplo:

{ display: grid; grid-template-columns: 1fr 1fr 1fr 1fr; font-size: 20px; }


De esta manera crear una distribución de un layout de 4 espacios a nuestra página y le damos un tamaño de letra de 20 megapixeles.

Selectores CSS

Selector de grupo


Un selector de grupo lo podemos identificar de la siguiente manera:

{ background-color: #blue;
font-size: 15px;
padding: 30px; }

De esta manera todas las etiquetas article que tengamos en nuestra página, se les aplicara los parámetros definidos para el color de fondo, tamaño de letra y espacio a ocupar.

Selector de clase


Un selector de clase lo declaramos por medio de (.) y le agregamos un nombre para identificarlo, lo podemos identificar de la siguiente manera:

.derecha { text-align: right;
color: #191C26; }

En este caso a todas las etiquetas que le definamos esta clase estarán alineados a la derecha y tendrán ese color definido.
Para asignarle a una etiqueta este selector lo hacemos de la siguiente manera.

<aside class = "derecha"> </aside>


Selector de identificación


Un selector de identificación lo declaramos por medio de (#) y le agregamos un nombre para identificarlo, lo podemos identificar de la siguiente manera:

#letra { font-size: 10px; }

En este caso a todas las etiquetas que le definamos esta identificación tendrán el tamaño de fuente aquí indicado.
Para asignarle a una etiqueta este selector lo hacemos de la siguiente manera.

<h5 id ="letra"> </aside>

Sintaxis CSS

La sintaxis en el lenguaje CSS se compone de 3 elementos, el primero es un selector el cual hace referencia a la etiqueta de HTML que se le aplicara el estilo, el siguiente elemento es la propiedad hace referencia a que propiedad de la etiqueta de HTML se le aplicara el estilo y por último el valor de la propiedad que se aplicara.
(Celaya Luna, 2014, p.14)

La propiedad o declaración se encierra entre las llaves o paréntesis y podrá contener una o varias propiedades, separadas por (;) cada propiedad debe ir acompañada por su valor establecido.
(Durango, 2015, p.12)

A un mismo elemento de HTML se le pueden asignar muchas propiedades definidas en CSS y estas pueden ser aplicadas a otros elementos de HTML.
(Tinoco, 2014, p.15)

Selectores CSS

Selectores de Grupo
Cuando se debe aplicar un estilo a mas de un elemento se pueden agrupar los selectores que comparten la misma definición, se escriben los selectores uno tras otro separados por una coma.
(Durango, 2015, p.15)

Selectores de Clase
Los selectores de clase permiten que se puedan establecer diferentes estilos que se le puedan aplicar a un mismo elemento. Adicionalmente se pueden crear selectores que se puedan usar en diferentes elementos del HTML.
(Durango, 2015, p.15,16)

Selectores de Identificación
Este selector es diferente al de clase porque se aplica a un solo elemento del código. Las reglas del HTML dictan que los atributos del selector no se pueden repetir en el código.
(Durango, 2015, p.18,19)
